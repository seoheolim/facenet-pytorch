# facenet-pytorch
facenet-pytorch 모델을 사용한 얼굴 모자이크 생성기

### How to install 
  - 가상환경 설치
1. pip install virtualenv 
2. virtualenv {venv_name} 
3. source {venv_name}/bin/activate
4. pip install -r requirements.txt
5. python main.py

### How to run
1. 인식에 사용할 영상을 test 디렉토리 안에 넣고, 파일명을 적절히 수정
2. python main.py
